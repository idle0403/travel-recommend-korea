`.env` 파일 생성:
```bash
OPENAI_API_KEY=sk-proj-your-key
GOOGLE_MAPS_API_KEY=your-google-key
```

### 2️⃣ Redis 시작 (선택적)

```bash
# Option A: Docker (추천)
docker run -d -p 6379:6379 redis:alpine

# Option B: 건너뛰기
# 자동으로 메모리 캐시 사용 (정상 동작)
```

### 3️⃣ 서버 시작

```bash
./start_server.sh
```
